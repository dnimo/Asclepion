## 🚀 使用建议

### 📦 安装方式
```bash
# 基础安装
pip install -e .

# 开发环境
pip install -e .[dev]

# 完整功能
pip install -e .[all]
```

### 🔧 开发环境设置
```bash
# 克隆项目
git clone https://github.com/dnimo/Asclepion.git
cd Asclepion

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 安装依赖
pip install -r requirements.txt
pip install -e .[dev]
```
